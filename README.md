### Overview
```sh
npm install -g canechair-cli

canechair-cli my-app 
cd my-app/
npm start
```


### What’s Inside?
* React/Redux/Thunk
* React-Router v4
* Webpack
* Babel
* EsLint/TsLint
* Jest
* Enzyme
* and Others

### Highlight
* TypeScripted ReactJS
* ES6 syntax support.
* Async api call.
* CI integrated for Travis and Appveyor
* When build for production:
  * Extract/Purify/Minify CSS
  * Uglify JavaScripts
  * Extract Bundles
